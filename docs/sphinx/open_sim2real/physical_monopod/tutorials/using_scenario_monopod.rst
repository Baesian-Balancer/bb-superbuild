======================
Using Scenario Monopod
======================

This tutorial will show how to use `scenario_monopod` package. This includes building
a cpp executable that reads and writes with the robot, additionally, the same thing
using python bindings. The goal of this tutorial will be to illustrate the versatility
of the scenario API and give a introduction into how the package can be used to create
custom application which interfaces into the robot. There will also be discussion on how
scenario can be easily switched into the ignition gazebo simulation backend.
